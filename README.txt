Created a BigInteger abstract data type in C++ that is capable of performing arithmetic operations on arbitrarily large signed integers. The underlying data structure for this ADT is the List of longs. 



Files:
	Arithmetic.cpp		Top level client for the project
	BigInteger.cpp		BigInteger implementation file
	BigInteger.h		BigInteger header file
	BigIntegerTest.cpp 	Test suite for BigInteger ADT
	List.cpp		Implementation of List ADT 
	List.h			Header file for List ADT
	ListTest.cpp		Test suite for List ADT
	Makefile		File to compile
	README.md 		Content listing of the project
